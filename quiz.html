JS Quiz

Что выведет alert?

В js имеются глобальные и локальные переменные. Что выведет каждый из alert-ов?

a = 1;
function f1 () {
    a = 2;
    alert(a);
}
function f2 (a) {
    a = 3;
    alert(a);
}
function f3 () {
    var a = 4;
    alert(a);
}

alert(a); // ?
f1(); // ?
f2(); // ?
f3(); // ?
alert(a); // ?

Выберите ответ:

Пояснение
a является глобальной переменной. Она при вызове функции f1 получает новое значение: 2. При вызове функции f2 в тело функции передается значение a в виде параметра. В этом примере мы аргумент явно не передаем, т.е. не вызваем функцию так: f2(55). Поэтому аргумент a функции f2 получает значение undefined, затем этому аргументу присваивается значение 3, что предотвращает переопределение глобальной переменной a. При вызове функции f3 переменная a является локальной, т.к. объявляется при помощи "var": var a = 4;.
